RustCast

Supercharged launcher for macOS. Fast, private, and fully open-source. Launch apps, search emojis, calculate, convert units, manage clipboard history, and run custom scripts—all from a sleek, customizable interface.

Lightning-Fast Launching

Instantly open applications with a single keystroke. No lag, no distractions. Built in Rust for unmatched speed and efficiency.

Endless Clipboard & Utilities

Infinite clipboard history keeps everything you copy. Quick emoji search, web opens, calculator, and unit conversions at your fingertips.

Fully Customizable

Tailor fonts, colors, and UI to your style. Switch modes like presentation for focused workflows. Run shell scripts seamlessly.

Privacy First, Zero Accounts

No sign-ups, no tracking. Everything stays local. Open-source code you can trust and contribute to.

RustCast Library

Powerful shell commands via the rustcast CLI. Extend functionality across your terminal and apps effortlessly.

How It Works

1. Install & Activate

Download RustCast 1.0. It's pre-activated—ready to use immediately, no purchases needed.

2. Summon with Shortcut

Hit Option + Space (or your custom hotkey). The launcher appears instantly.

3. Search & Execute

Type to launch apps, search, calculate, or run scripts. Results update in real-time.

4. Customize & Extend

Adjust themes, add modes, or integrate the RustCast library for advanced automation.
